# cidadania-portuguesa
Repositório para testes de aplicação de IA generativa para responder perguntas sobre a obtenção de nacionalidade portuguesa.

#Embeddings

Embeddings são representações numéricas (vetores) de textos  (ou outros objetos, como imagens, áudio, etc). Eles servem para que os modelos de aprendizado de máquina encontrem objetos semelhantes utilizando pesquisas de similaridade.
